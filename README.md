# Resume Screening System (NLP Project)

## 📌 Project Overview
This project screens resumes automatically by matching them with a Job Description (JD).

It ranks resumes based on similarity score using:
- TF-IDF + Cosine Similarity
- Sentence Embeddings + Cosine Similarity

Also includes skill gap analysis.

---

## 🚀 Features
✅ Upload multiple resumes (PDF/DOCX)  
✅ Paste Job Description  
✅ Resume Ranking  
✅ Similarity Score Output  
✅ TF-IDF and Embedding Matching  
✅ Threshold Filtering  
✅ Download results as CSV  
✅ Skill Gap Analysis (missing skills detection)
